**Smart To-Do List Web Application**

A simple, responsive, and modern To-Do List website built using HTML, CSS, and JavaScript.
This project allows users to create, manage, and track their daily tasks efficiently.
All tasks are saved using LocalStorage, so the data remains even after refreshing the page.

_Features:_
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks (All / Active / Completed)
- Saves tasks automatically using LocalStorage
- Fully responsive UI
-Clean and modern design

_Technologies Used:_
HTML5-Structure of the webpage
CSS3-Styling + Responsive design
JavaScript-Application logic
LocalStorage-Client-side data storage
